React-native Windows 7 环境准备(android)

Posted on 2017-02-10   |   In 经验总结   |  

提前准备

软件名称 地址 备注
Node.js nodejs.cn 主要用于下载react-native node_modules模块插件等
Python Python 2.7 react-native 某些模块需要调用python
Android Studio Android Studio 2.0 运行以及打包安卓app应用
Git Git 代码版本管理
Genymotion Genymotion 性能比较好的安卓模拟器
VirtualBox VirtualBox 运行模拟器的虚拟机

安装步骤

  • 具体可参考 ：http://reactnative.cn/docs/0.41/getting-started.html#genymotion

遇到的坑

  • Genymotion 在国内一般会遭到墙，因此需要配置代理’

‘打开Genymotion，并找到 setting ‘

‘设置代理端口以及账号名密码 ‘

  • 全部准备玩之后，在起动也需要注意

1.优先执行

2.等模拟器已经打开且进入系统后则执行

  • 当起动后第一次打开会出现红屏情况

1.调起虚拟机设置菜单按钮， ctrl+d

2.设置模拟器代理IP 端口，设置为react-native start 起动的ip和端口

3.设置完成后再调起虚拟机设置菜单按钮，然后点击reload 则完成
